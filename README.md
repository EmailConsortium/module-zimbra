# Module quản lý tài khoản mở rộng cho zimbra
Các yêu cầu chính:
- Khởi tạo tài khoản trên NukeViet Portal là có thể khởi tạo luôn tài khoản Zimbra.
- SSO giữa NukeViet và Zimbra bằng chính tài khoản đã đăng ký.
